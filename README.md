# 📲 WhatsApp Automation Server

Servicio backend encargado de la automatización de mensajes mediante WhatsApp, integrado con sistemas de gestión de turnos.

## 🚀 Funcionalidades
- Envío automático de confirmaciones de turnos
- Recordatorios programados (24h y 3h antes)
- Integración con sistemas externos (barbería / beauty)
- Manejo de lógica de mensajes y estados

## 🛠️ Tecnologías
- Node.js
- JavaScript
- API de WhatsApp

## ⚙️ Arquitectura
- Servicio independiente de mensajería
- Conexión con backend principal
- Automatización de eventos

## 🔗 Integración
Este servicio se conecta con:
- beauty-eyes-app
- barbería (sistema de turnos)
